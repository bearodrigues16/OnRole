O que é o Search On Role?

Search on Rolê é uma aplicação mobile desenvolvida com a finalidade de gerenciar os rolês de um modo geral, seja como organizadores, prestadores de serviços ou demais usuários em busca de rolês, visando melhorar a qualidade dos rolês e, dessa forma, proporcionar uma maior interação entre seus diferentes usuários.
